The WCU Student Assistant Bot is an intelligent Telegram bot designed to help Wachemo University students access their academic information quickly, securely, and conveniently. The bot allows students to:

Register securely using their WCU student ID.

View their profile with encrypted and safe personal data.

Access grade reports after password verification.

Delete their account with a fun math verification for security.

Receive privacy-conscious interactions, ensuring student data is encrypted and never stored in plaintext.

This bot provides an interactive, user-friendly experience on Telegram, replacing the need for students to manually navigate multiple portals. It also demonstrates practical applications of encryption, database management, and conversational bot design in a real-world university context.
